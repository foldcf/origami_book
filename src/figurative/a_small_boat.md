## A SMALL BOAT

<img src="./a_small_boat_19.jpg" alt="a_small_boat_19" width="75%" height="75%">

***
### ABOUT THE MODEL

I developed this model in June 2023. It is a simple one. I recommend using thick paper to allow wet-folding.

***

### STEPS

|                                                                                                                                                                            |                                                                                                                                                                  |
| :------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <img src="./a_small_boat_1.jpg" alt="a_small_boat_1" width="140%" height="140%">                                                                                           | Start with a rectangular sheet of paper.                                                                                                                         |
| <img src="./a_small_boat_2.jpg" alt="a_small_boat_2" width="140%" height="140%">                                                                                           | Make alternating mountain and valley folds.                                                                                                                      |
| <img src="./a_small_boat_3.jpg" alt="a_small_boat_3" width="140%" height="140%">                                                                                           | Fold into an accordion shape.                                                                                                                                    |
| <img src="./a_small_boat_4.jpg" alt="a_small_boat_4" width="140%" height="140%">                                                                                           | Get the center, unfold.                                                                                                                                          |
| <img src="./a_small_boat_5.jpg" alt="a_small_boat_5" width="140%" height="140%">                                                                                           | Towards one side (from the center) fold the model into a C shape.                                                                                                |
| <img src="./a_small_boat_6.jpg" alt="a_small_boat_6" width="140%" height="140%">                                                                                           | Repeat on the other side.                                                                                                                                        |
| <img src="./a_small_boat_7.jpg" alt="a_small_boat_7" width="140%" height="140%">                                                                                           | Unfold.                                                                                                                                                          |
| <img src="./a_small_boat_8.jpg" alt="a_small_boat_8" width="140%" height="140%">  <img src="./a_small_boat_9.jpg" alt="a_small_boat_9" width="140%" height="140%">         | Convert all the zigzags formed in step 5 and 6 into mountain folds. You'll notice that the boat is slowly taking shape. Continue reinforcing the mountain folds. |
| <img src="./a_small_boat_10.jpg" alt="a_small_boat_10" width="140%" height="140%"> <img src="./a_small_boat_11.jpg" alt="a_small_boat_11" width="140%" height="140%">      | Notice that once a mountain fold crosses a zigzag, it changes to a valley fold; and a valley fold becomes a mountain fold.                                       |
| <img src="./a_small_boat_12.jpg" alt="a_small_boat_12" width="140%" height="140%"> <img src="./a_small_boat_13.jpg" alt="a_small_boat_13" width="140%" height="140%">      | Push the three central mountain folds downwards.                                                                                                                 |
| <img src="./a_small_boat_14.jpg" alt="a_small_boat_14" width="140%" height="140%">  <img src="./a_small_boat_14_5.jpg" alt="a_small_boat_14_5" width="140%" height="140%"> | Spray some water in the regions circled and tie with a rubber band/clip. Allow it to dry.                                                                        |
| <img src="./a_small_boat_15.jpg" alt="a_small_boat_15" width="140%" height="140%">                                                                                         | Done.                                                                                                                                                            |
